HTML5

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web.

Tabela HTML

Elementos Semânticos Usei ou Não Descrição
header ✔ Usei em todos para separar o titulo e a sua frase introdutoria
nav ✔ Usei na parte onde seleciona-se informação,multimedia,etc ficava mais claro e organizado
main ✔ Usei em todos pela sua organização
article ❌ Não achei que fosse preciso
section ✔ Usei em tudo o que podia dentro do main tambem usei em tabelas para deixar o codigo mais claro e percetivel
aside ❌ Não usei porque nao usei anotações em lado nenhum iria separar informações sobre o site e iria ficar confuso
footer ✔ Usei para dizer que era o autor

CSS

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.

Tabela CSS
